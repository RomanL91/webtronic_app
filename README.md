Все что Вам нужно: 
1) Создать файл .env на уровне с Dockerfile
2) Внести в него следующее:
  - MONGO_URL='mongodb://localhost:27017'
  - JWT_SECRET='mytoken'
  - ALGORITM='HS256'
3) Docker-compose build, 
4) Docker-compose up
